# One Minute Timer

A simple timer that counts down one minute.

## Run It

```
python -m SimpleHTTPServer
open locahost:8000
```

Then click "Start".

## Development

```
chmod +x install.sh
./install.sh
```

This downloads the latest version of [tock.js](https://github.com/mrchimp/tock).
